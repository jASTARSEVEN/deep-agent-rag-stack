# qmsum-query-summary-curated-pilot-v1

## Purpose

This package provides a small English query-conditioned summary pilot package for the bilingual summary/compare benchmark lane.

## What's Included

- `manifest.json`
- `questions.jsonl`
- `source_documents/`
- `reference_run_summary.json`

## Notes

- Retrieval scope is defined as `explicit_document_ids` and resolved from `document_file_names` at benchmark run time.
